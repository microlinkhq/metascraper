> metascraper integration with Clearbit Logo API.

## Install

```bash
$ npm install metascraper-clearbit --save
```

## API

### clearbitLogo([options])

#### options

##### size

Type: `string`<br>
Default: `128`

Length of longest side in pixels.

##### format

Type: `string`<br>
Default: `png`

Image format, either "png" or "jpg".

##### greyscale

Type: `boolean`<br>
Default: `false`

Desaturates image if set to `true`.
