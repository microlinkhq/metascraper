## Install

```bash
$ npm install metascraper-date --save
```
## API

### metascraper-date([options])

#### options

##### datePublished

Type: `boolean`<br>
Default: `false`

Whether to return `datePublished` alongside date.

##### dateModified

Type: `boolean`<br>
Default: `false`

Whether to return `dateModified` alongside date.
